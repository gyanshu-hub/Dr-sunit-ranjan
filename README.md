# Dr. Sandip Kumar Gupta – Healthcare Website

A modern, clean, and professional static website for **Dr. Sandip Kumar Gupta**, a General Physician based in **Bhorey, Bihar, India**.

---

## 🏥 About the Project

This is a fully responsive, mobile-friendly healthcare website designed with a calming medical theme (white, light blue, and green). It provides patients with all the information they need to connect with the doctor and book appointments.

---

## ✅ Completed Features

### Pages / Sections
- **Hero Section** — Doctor name, tagline "Trusted Healthcare for You & Your Family", CTA buttons (Book Appointment, Call Now), hero stats
- **About Doctor** — Introduction, qualifications, experience, expertise, languages spoken
- **Services** — 6 service cards with icons: General Consultation, Diagnosis & Treatment, Preventive Healthcare, Child Healthcare, Chronic Disease Care, Emergency First Aid
- **Appointment Section** — Booking form (Name, Phone, Date, Time Slot, Reason), WhatsApp CTA
- **Testimonials** — 5 realistic patient reviews with star ratings
- **Contact Section** — Address, phone number, clinic hours, WhatsApp link, contact form, Google Maps embed
- **Footer** — Brand info, quick links, service links, contact info, copyright notice

### Navigation & UX
- Sticky header with active link highlighting on scroll
- Smooth scroll navigation
- Mobile hamburger menu with overlay
- Active nav link tracking

### Mobile & Accessibility
- Fully responsive for mobile, tablet, and desktop
- Floating WhatsApp button
- Floating "Call Now" button for mobile users
- ARIA labels on interactive elements
- Alt texts and semantic HTML

### Animations & Interactivity
- Fade-in animations (up, left, right) triggered on scroll
- Hover effects on cards and buttons
- Counter animation for hero stats
- Form validation with inline error feedback
- Appointment form → WhatsApp redirect on submit
- Contact form success state

### SEO & Performance
- Local SEO meta tags (Bhorey, Bihar keywords)
- Open Graph tags for social sharing
- Semantic HTML5 structure
- Fast-loading CDN resources (Google Fonts, Font Awesome)
- Lazy-loaded Google Maps embed
